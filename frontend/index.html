🔌 Endpoints da API

GET /api/usuarios
Listar todos os usuários
POST /api/usuarios
Criar novo usuário
GET /api/veiculos
Listar todos os veículos
POST /api/veiculos
Cadastrar novo veículo
GET /api/motoristas
Listar todos os motoristas
POST /api/motoristas
Cadastrar novo motorista
GET /api/cargas
Listar todas as cargas
POST /api/logistica/registrar-carga
Processo completo (carga + rota)
GET /api/entregas
Listar todas as entregas
PUT /api/entregas/{id}/status
Atualizar status da entrega
GET /api/dashboard
Dados completos do dashboard
GET /api/dashboard/metricas-rapidas
Métricas rápidas
